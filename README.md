# Seif Medhat - Integrating With HubSpot I: Foundations Practicum

## About
This project was completed by **Seif Medhat** as part of the HubSpot Academy course **Integrating With HubSpot I: Foundations Practicum**.

The app is a simple Node.js + Express + Pug web app that connects to the HubSpot CRM API.
It creates and lists records for a custom object called **Video Games** with these properties:
- name (string)
- publisher (string)
- price (number)
